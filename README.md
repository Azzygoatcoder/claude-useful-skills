# Claude Useful Skills

Claude Code 实用技能集合。每个子目录是一个独立可安装的插件。

## 包含的插件

| 插件 | 目录 | 版本 | 说明 |
|------|------|------|------|
| Code Security Skills | [code-security-skills/](code-security-skills/) | v1.3.0 | FABLE-5 风格指令重写：门控、信心标注、自检门、元认知陷阱 |

## 安装

每个插件独立安装。以 code-security-skills 为例：

```bash
claude plugins install https://github.com/Azzygoatcoder/claude-useful-skills --path code-security-skills
```
